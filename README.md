# mini-shop 🛍️

> 동시성 제어와 분산 트랜잭션 문제 해결을 위한 쇼핑몰 토이 프로젝트  


## 📌 프로젝트 개요
mini-shop은 기본적인 쇼핑몰 도메인(상품 관리, 주문, 알림)을 바탕으로  
**동시성 제어(재고 차감)**와 **분산 트랜잭션(주문-알림 보장)** 문제를 다루기 위해 설계된 프로젝트입니다. 


## 🛠️ 기술 스택
- **Backend**: Java 21, Spring Boot 3.5.6, Spring Data JPA
- **DB**: MySQL, Redis
